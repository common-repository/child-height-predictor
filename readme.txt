How tall will your kid grow? Predict the height of your child!

== Description ==

Child Height Predictor us a Wordpress Plugin that predicts the height of a child based on the height of their parents.

Features include:

* Add to pages and posts using the shortcode [childheight]
* Add to sidebars by adding the Child Height Predictor Widget
* Localize it to your language by using [http://www.poedit.net/](http://www.poedit.net/ "poedit") to translate the added .po file
* Change from metric to us system.

See the [Child Height Predictor site](http://www.ostheimer.at/wordpress-plugins/child-height-predictor/ "Child Height Predictor site") for a test run!

== Installation ==

1. Upload  "child-height-predictor" folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add the Childheight Calculator Widget to your widget enabled space
1. If you need insert in article please use short tag [childheight].
